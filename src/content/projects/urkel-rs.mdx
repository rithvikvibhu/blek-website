[Urkel](https://github.com/handshake-org/urkel) is an optimized and cryptographically provable key-value store. It is a sparse Merkle tree that powers the Handshake chain.

`urkel-rs` is a Rust port/rewrite of the whole library (that is written in JavaScript).
