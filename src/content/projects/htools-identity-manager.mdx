An Identity Manager that uses [`handshake-login`](https://github.com/htools-org/handshake-login) and works with all supported strategies.

A drop-in replacement for Namebase's ID Manager. Built from scratch to support multiple strategies.

Allows authenticating with any handshake name you own with DNS.

For more info:

- [Namebase Learning Center - Handshake-based OIDC Authentication Protocol](https://learn.namebase.io/development/guides/handshake-login/handshake-based-oidc-authentication-protocol)
