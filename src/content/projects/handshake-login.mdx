A TypeScript library for authentication with Handshake names, for use by validating servers (websites) and identity managers (clients/browsers/etc).

## Features

- Add **Login with Handshake** to your website
- Build Identity Managers
- Use without any intermediaries (OAuth servers, etc.)
- Works in NodeJs and Browsers
- Supports multiple strategies (Local Storage, WebAuthn)
