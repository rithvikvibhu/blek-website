[Handshake](https://handshake.org) is a decentralized naming system for the web. It follows [BIP-9](https://github.com/bitcoin/bips/blob/master/bip-0009.mediawiki) for network upgrades and consensus changes.

This website tracks the status of BIP-9 signalling of soft forks and hard forks.
